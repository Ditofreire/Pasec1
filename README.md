# Portafolio web de ciudadanía responsable

## Archivos principales

- `index.html`: contenido de la página.
- `styles.css`: diseño visual.
- `script.js`: menú para celulares.
- `documentos/ciudadania-responsable-inicial.pdf`: PDF inicial incrustado.
- `assets/producto-ia.jpg`: fotografía opcional del producto mejorado.

## Cómo abrirla

Haz doble clic en `index.html`.

## Cambios importantes antes de entregar

1. En `index.html`, cambia `[Semestre]` si deseas agregarlo junto a tu carrera.
2. Añade una fotografía con el nombre `producto-ia.jpg` dentro de `assets`.
3. Sube tu video a YouTube.
4. Reemplaza `REEMPLAZA_ESTE_ID` por el ID real del video.
5. Revisa los compromisos y ajusta cualquier dato que no represente exactamente tus decisiones.

## Cómo publicar gratis con GitHub Pages

1. Crea una cuenta o inicia sesión en GitHub.
2. Crea un repositorio nuevo, por ejemplo `portafolio-ciudadania`.
3. Sube todos los archivos manteniendo las carpetas.
4. En el repositorio abre `Settings` → `Pages`.
5. En `Build and deployment`, selecciona `Deploy from a branch`.
6. Elige la rama `main` y la carpeta `/root`.
7. Guarda. GitHub generará el enlace público.
